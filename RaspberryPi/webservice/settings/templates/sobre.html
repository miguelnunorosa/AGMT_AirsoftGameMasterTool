Sobre o projeto

O presente sistema pretende registar todos os participantes/utilizadores dos modos de jogo desenvolvidos. Além deste registo, o sistema guarda também os resultados dos jogos para processar pequenas estatísticas dos mesmos.

Atualmente apenas está disponível o jogo de "Desarmar a Bomba". Em breve irão estar disponíveis mais funcionalidades.

Funcionalidades a implementar:

  • Selecionar um jogo da lista e enviar para o dispositivo eletrónico para iniciar o jogo
  • Configuração do jogo via web/bluetooth (app mobile) e enviar para o dispositivo
  • Formar equipas com os jogadores ativos registados
